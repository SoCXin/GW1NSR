This example demonstrates SPI communication with PCD8544 "Nokia 5110 LCD".

 --------------------                   ------------------------
|                    |                 |                        |
|                PC3 |---------------->|1 RST                   |
|                PC4 |---------------->|2 CE                    |
|                PD2 |---------------->|3 DC                    |
|                PC6 |---------------->|4 DIN                   |
|       <STM8>   PC5 |---------------->|5 CLK  <5110 LCD>       |
|                    |          [3V3]<-|6 VCC                   |
|                    |             ----|7 LED                   |
|                    |            |----|8 GND                   |
|                    |           _|_   |                        |
|                    |          [GND]   ------------------------
 --------------------
